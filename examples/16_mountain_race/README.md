# Mountain race

Simple game with different levels represents racing in the mountains. First level is the day time and the second level is the night time.

## Usage:
```bash
  $ make
  $ ./mountain_race levels/01.lvl
  $ ./mountain_race levels/02.lvl
  ...
```
